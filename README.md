Lightweight PDF Viewer
======

Tools used: tkinter,auto-py-2-exe

Installation:
======

Windows:

Step 1: Run the EXE

Running Python Source Code:
======

Step 1: Clone the repo

Step 2: CD into the repo

Step 3: Run "python .\pdfviewer.py"
